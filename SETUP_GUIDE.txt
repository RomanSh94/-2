                        X20 BOT — БЫСТРЫЙ СТАРТ

📋 ТРЕБОВАНИЯ:
  • Windows 10+ / macOS / Linux
  • Python 3.12 или 3.13 (3.14 может вызвать проблемы)
  • Telegram аккаунт
  • OpenAI API ключ (https://platform.openai.com/api-keys)

═══════════════════════════════════════════════════════════════════════════
                           WINDOWS ПОЛЬЗОВАТЕЛИ
═══════════════════════════════════════════════════════════════════════════

1. СКАЧАЙТЕ И УСТАНОВИТЕ Python 3.12:
   https://www.python.org/downloads/release/python-3122/
   
   При установке ОБЯЗАТЕЛЬНО отметьте:
   ✅ "Add Python 3.12 to PATH"

2. ОТКРОЙТЕ PowerShell В ПАПКЕ ПРОЕКТА

3. СОЗДАЙТЕ ВИРТУАЛЬНОЕ ОКРУЖЕНИЕ:
   
   python -m venv venv
   .\venv\Scripts\Activate.ps1

4. УСТАНОВИТЕ ЗАВИСИМОСТИ:

   pip install --upgrade pip
   pip install -r requirements.txt

5. СКОПИРУЙТЕ И ОТРЕДАКТИРУЙТЕ .env:

   copy .env.example .env
   notepad .env
   
   Заполните:
   BOT_TOKEN=ваш_токен_телеграма
   OPENAI_API_KEY=ваш_openai_ключ
   ADMIN_PASSWORD=надёжный_пароль

6. ЗАПУСТИТЕ БОТ:

   python bot.py

═══════════════════════════════════════════════════════════════════════════
                         LINUX / macOS ПОЛЬЗОВАТЕЛИ
═══════════════════════════════════════════════════════════════════════════

1. УБЕДИТЕСЬ, ЧТО УСТАНОВЛЕН Python 3.12:

   python3 --version

2. ПЕРЕЙДИТЕ В ПАПКУ ПРОЕКТА И СОЗДАЙТЕ ОКРУЖЕНИЕ:

   python3 -m venv venv
   source venv/bin/activate

3. УСТАНОВИТЕ ЗАВИСИМОСТИ:

   pip install --upgrade pip
   pip install -r requirements.txt

4. СКОПИРУЙТЕ И ОТРЕДАКТИРУЙТЕ .env:

   cp .env.example .env
   nano .env
   
   Заполните обязательные поля (см. выше)

5. ЗАПУСТИТЕ БОТ:

   python3 bot.py

═══════════════════════════════════════════════════════════════════════════
                             ГДЕ НАЙТИ КЛЮЧИ
═══════════════════════════════════════════════════════════════════════════

BOT_TOKEN (Telegram):
  1. Откройте Telegram и найдите @BotFather
  2. Команда: /newbot
  3. Назовите бота (например, "X20")
  4. Получите токен вида: 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

OPENAI_API_KEY:
  1. Откройте https://platform.openai.com/api-keys
  2. "Create new secret key"
  3. Скопируйте ключ (он больше не будет виден!)

═══════════════════════════════════════════════════════════════════════════
                           ПОСЛЕ ЗАПУСКА БОТА
═══════════════════════════════════════════════════════════════════════════

✅ Telegram:
  Откройте чат с вашим ботом и напишите /start

✅ Admin Dashboard:
  http://localhost:8080
  Логин: admin
  Пароль: то что вы указали в ADMIN_PASSWORD

═══════════════════════════════════════════════════════════════════════════
                          ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
═══════════════════════════════════════════════════════════════════════════

1. Проверьте правильность токенов в .env
2. Убедитесь, что виртуальное окружение активировано
3. Прочитайте логи в консоли (там обычно есть ошибка)
4. Проверьте интернет-соединение
5. Попробуйте переустановить зависимости:
   pip install --force-reinstall -r requirements.txt
